# QuickShopLimited
自用，限制玩家在单个商店内出售、收购物品的最大数量。

## 命令
/qs limit <amount> - 设置单玩家最大交易数量限制，-1为重置为初始不限制状态并清空记录的交易数据  
  Permission node: quickshop.limited
